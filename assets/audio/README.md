# Audio runtime assets

Drop runtime audio files here using the canonical names expected by `src/assets/assetContract.js` and `src/systems/AudioHooks.js`.

## Expected files

- `sfx_attack.wav`
- `sfx_enemy_attack.wav`
- `sfx_hit.wav`
- `sfx_enemy_hurt.wav`
- `bgm_main.ogg` (optional)

If files are missing, the game falls back to synthesized tones for core combat cues.
